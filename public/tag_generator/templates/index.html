Sim Log Comparison

For each saved sim run, this harness runs every build-path algorithm on the same hero / team comp / build, then scores how closely each algo's final inventory matches what you actually bought. Bucket weights: win/good=+5, loss/good=+4, win/neutral=+3, loss/neutral=+2, win/bad=+1, loss/bad=−2.

        Paste a build JSON exported from any Hero Edit page (format:
        { heroKey, buildName, build }). Choose where it lands —
        the build is appended to that hero's builds in the target pack, or
        replaces an existing build with the same name.
